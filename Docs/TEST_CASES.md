# Проверочные случаи

## 1. Кромка не меняется при повороте

Деталь:

```text
Размер: 800×400
Количество: 2
Кромка по L1: 2 мм
Кромка по L2: 2 мм
Кромка по W1: нет
Кромка по W2: 0.4 мм
```

Ожидание:

- общий метраж кромки одинаковый независимо от поворота;
- на визуализации стороны кромки должны повернуться вместе с деталью.

## 2. Пропил не прибавляется к каждой детали

Настройки:

```text
Пила: 4 мм
Лист: 1000×1000
Детали: 2 штуки 400×400
```

Ожидание:

- между двумя соседними деталями учитывается один фактический пропил 4 мм;
- программа не делает каждую деталь условно 404×404 со всех сторон.

## 3. Остаток используется первым

Листы:

```text
Остаток: 1000×2800, приоритет 1
Стандарт: 1500×2800, приоритет 2
```

Ожидание:

- программа сначала пытается разместить детали на остатке;
- стандартный лист используется только после невозможности или исчерпания остатка.

## 4. Деталь не помещается без поворота

Лист:

```text
1000×2800
```

Деталь:

```text
1200×500
Поворот запрещён
```

Ожидание:

- программа не размещает деталь;
- выдаёт ошибку: деталь не помещается без поворота.

## 5. Деталь помещается с поворотом

Лист:

```text
1000×2800
```

Деталь:

```text
1200×500
Поворот разрешён
```

Ожидание:

- программа пробует поворот;
- размещает как 500×1200;
- кромка остаётся привязанной к логическим сторонам детали.

## 6. Отступы от краёв листа

Лист:

```text
1000×1000
```

Отступы:

```text
слева 10
справа 10
сверху 10
снизу 10
```

Ожидание:

- рабочая область становится 980×980;
- детали не выходят за эту область.

## 7. Размер после кромки

Деталь:

```text
Чистовой размер: 397×397
Кромка: 1 мм с двух сторон
Прифуговка: 0.5 мм с двух сторон
```

Ожидание:

```text
Размер без кромки:
397 - 1 - 1 = 395

Размер заготовки / распиловочный размер:
395 + 0.5 + 0.5 = 396
```

## 8. Неразмещённые детали

Если часть деталей не помещается, программа должна:

- не молчать;
- не терять детали;
- вывести список неразмещённых деталей;
- добавить предупреждение в Excel.
